Hire the Top 1% Without the Noise

An AI-powered recruitment platform that analyzes resumes against job descriptions, evaluates GitHub coding quality, and ranks candidates with transparent, explainable scoring across 6 skill dimensions.

! The Problem

  • Manual resume screening is slow and biased toward keywords
  • No way to verify actual coding quality from a resume alone
  • Traditional ATS systems are opaque -- no explanation of rankings
  • Bulk candidate processing requires manual effort per candidate
  • Scanned/image-based resumes are ignored by most tools

✓ Our Solution

  • AI agent autonomously analyzes CVs with contextual understanding
  • GitHub Vibe Coding Score detects AI-generated vs. human-written code
  • Transparent "Why This Rank?" explanations for every candidate
  • CSV bulk import with per-row AI analysis in a single upload
  • OCR fallback for scanned PDFs and image-based resumes
Key Features

🧠 Agentic AI Analysis

Powered by Gemini 2.5 with autonomous tool calling. The AI decides which tools to invoke (GitHub repos, coding analysis, profile stats, CV evaluation) in a loop until it has complete insight.

📊 6-Axis Radar Scoring

Each candidate is scored across Technical, Experience, Education, Soft Skills, Culture Fit, and Growth -- displayed as an interactive radar chart for instant visual comparison.

🔍 Vibe Coding Detection

Analyzes GitHub repositories to determine how much code is AI-generated vs. hand-written. Smart file sampling picks the 5 most meaningful source files per repo for accurate analysis.

🔗 Auto GitHub Discovery

Automatically extracts GitHub profile URLs from hyperlink annotations embedded in PDF resumes -- candidates don't need to manually specify their GitHub profiles.

📄 Multi-Format Support

Upload PDF, DOCX, or image files. Scanned PDFs automatically fall back to Tesseract OCR. CSV bulk import processes entire candidate batches with full AI analysis per row.

💬 Explainable Rankings

Every candidate gets a "Why This Rank?" narrative and detailed AI summary. Full transparency into the scoring logic -- no black-box decisions.

AI Analysis Pipeline

⚙ Dual-Model Agentic System with Tool Calling

Gemini 2.5 Flash (Primary Agent)

  • Autonomous tool-calling loop
  • Extracts skills, experience, education, projects
  • Invokes GitHub tools to fetch repos & analyze code
  • Generates matching score, category scores & summary

Gemini 2.5 Pro (Evaluator)

  • Higher-accuracy model for CV-job matching
  • Evaluates: Skills Match, Experience Relevance, Education Fit
  • Returns overall_fit assessment & calibrated matching_score
  • Called as a tool by the primary agent when needed
